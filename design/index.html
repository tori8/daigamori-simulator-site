焼く前に、色がわかる。アルファ版

台ヶ森焼 陶器色味シミュレーター 設計文書 — このアプリが「何をするものか」「どう動いているか」を、図解で大きな話から細かい話へ掘り下げます。

課題

陶器の色は、焼き上がるまで誰にもわからない。

同じ形でも「土」「釉薬(うわぐすり)」「焼き方」の組み合わせで、仕上がりの色も表情もまったく変わります。試作には材料費と窯の時間がかかり、全部を試すことはできません。

解決

その組み合わせを、画面上で試せるようにした。

購入を検討する方が好みの組み合わせを探したり、作り手との相談の たたき台にできるWebアプリです。スマホでも使えます。

図1: 全体像。1〜3が画面で体験できる部分、4は精度を上げるための開発工程

選べる組み合わせ

3土(素地)
8釉薬
8形状
2×3焼成方式 × 焼き方
1,152通りの組み合わせ
① 土(素地)赤土 / 唐土 / 白磁
② 釉薬透明釉 / 灰釉 / マット釉 / 結晶釉 / 青磁釉 / 鉄釉 / 緑釉 / 織部釉
③ 形状皿 / 湯呑 / 急須 / ぐい飲み / 徳利 / 茶碗 / 植木鉢 / 丼
④ 焼成方式酸化 / 還元 — 窯の中の空気の状態。色が大きく変わる
⑤ 焼き方薪 / ガス / 電気 — 窯の種類。薪は偶然の変化が豊か

この全部を試作するのは現実的でないからこそ、シミュレーションに意味があります。
